# vs-pipeline-mod
A mod for Vintage Story introducing pipelines and pumps, as well as various applications for such devices.

Current parts:
* Pipe pieces
* Pump (mechanical network)
* Sprinkler
* Water drain (inlet)
* Pipe mold & shaping tool
